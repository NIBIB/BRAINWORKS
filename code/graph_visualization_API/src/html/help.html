Instructions

Moving

  • Click and drag an empty space to move the viewing window.
  • Click the Center button to move back to the default position and zoom level.

Zooming

  • Use the scroll wheel to expand the graph in and out.
  • Hold ctrl while scrolling to zoom without expanding.

Searching

  • Use the search bar to highlight all nodes whose name contains that query.
  • Click the button to the left of the search bar to switch between searching for nodes or edges.
  • Enter names separated by commas to search for multiple nodes or edges.
  • Click away from the search bar to clear the current search.

Organization

  • Click and drag a node to change its position.
  • Click the Organize button to optimize the arrangement of nodes.

Node Connections

  • Left-click a node to highlight its immediately adjacent neighbors.
  • Left-click a second node to highlight the shortest path between the two, if one exists.

Cluster Tree (if available)

  • Click the + or - buttons to expand and collapse all clusters.
  • Double-click a node to expand it if it is a cluster.

Download

  • Click the Download button to download the current graph in JSON format.

Upload

  • Click the Upload button to upload a properly formatted JSON graph of your own.
